# `fdroid-setup`

Sets up or starts a local F-Droid build environment.

**Usage**:

```console
$ fdroid-setup [OPTIONS] YAML_FILE
```

**Arguments**:

* `YAML_FILE`: F-Droid metadata file (ie. com.example.myapp.yml)  [required]

**Options**:

* `--gitlab-user TEXT`: GitLab username for your fdroid-data fork  [env var: GITLAB_USER]
* `--install-completion`: Install completion for the current shell.
* `--show-completion`: Show completion for the current shell, to copy it or customize the installation.
* `--help`: Show this message and exit.
